## System Description

A Student Management System is used to store and manage student records.
It helps schools track students, subjects, and grades (easily).
Everything is saved in one database.

## ⚙️ Key Functionalities
	•	Add, edit, delete student info
	•	Enroll students in subjects
	•	Record grades
	•	View student details
	•	Search student records


## 👥 Stakeholders
	•	Students
	•	Teachers
	•	Registrar
	•	School Admin


## 🔄 Core Data Processes
	•	Input student data
	•	Save to database
	•	Update records
	•	Retrieve/search info
	•	Generate reports
